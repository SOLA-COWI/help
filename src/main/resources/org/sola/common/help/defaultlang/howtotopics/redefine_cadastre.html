Redefine Cadastre

The Redefine Cadastre screen allows you to change the boundaries of an existing cadastre object both entirely and point by point (adding nodes and changing nodes coordinates). It can be accessed by starting a Redefine Cadastre service from the Application Details Service tab



Also this screen extends the SOLA Map Viewer with additional tools to aid spatial editing of parcels. The additional tools provided are:

  • Add node
  • Select node to modify
  • Reset changes
  • Select a boundary to change
  • Change selected boundary
  • Add image
  • Remove added image

Nodes

Nodes represent crucial elements in the boundary. New nodes can be added using the Add new node tool and existing ones can be selected using the Select a Node to Change tool



The coordinates of new or existing node can be then manually changed entering them directly into the opened dialog box and clicking modify. To remove the node click Remove



The new boundary will appear in light green while the old one will be in dark green thus pointing out the differences between the two.



Once modified the coordinates or removed the node, the cadastre definition process can be reset using the Reset tool . When the redefinition is complete, Save the cadastre redefinition. To complete the cadastre redefinition, complete the Redefine Cadastre service using the Complete tool on the Application Details Services tab and then approve the application. This will cause the new boundary to become current and the old to become historic.
