Property Details

The Property Details screen allows you to create new property records or edit the details of an existing property. You can access the Property Details screen by starting a property related service such as New Digital Title (used to create new property records), Register New Ownership, Register Mortgage, Vary Lease, etc, from the Application Details Service tab.



A property in the context of the Land Administration Domain Model (LADM - draft ISO standard 19152) is a Basic Administrative Unit (a.k.a. BA Unit). In a title registration system, it is the title. In line with LADM, the BA Unit is uniquely identifiable and links parcels to their applicable Rights, Restrictions and/or Responsibilities (a.k.a. RRR) for the purposes of registration with a land administration agency. Details that can be captured for a property are:

  • Parcels – The area(s) of land that define the extent of a property. Parcels can be spatially defined and/or textually described.
  • Rights, Restrictions & Responsibilities – The rights, restrictions or responsibilities the right holder(s) are accorded in relation to the parcels represented by the property. These could be ownership rights, lease rights, mortgage rights, mining rights, access rights, servitudes, building restrictions, land use restrictions, caveats, etc.
  • Ownership – Typically the primary right for a property is the right of ownership. Properties can have one or many owners each with a share in the property that may be implicitly defined or undivided. Note that changes in ownership are captured using the Ownership right on the Rights / Restrictions tab of the Property Details screen. The Ownership tab is provided on the Property Details screen to quickly view ownership history.
  • Notations – Short comments recorded when changes are made to the property that provide a succinct history of the changes.

Applications with Multiple Properties

SOLA allows more than one property to be associated to an application. In this situation, the Select Property screen is displayed when you start a property related service allowing you to select the appropriate property to work on by double clicking.



Approving a Property Change

Changes made to the property such as the addition or removal of rights remain pending until the application is approved. You can approve the application by completing all services on the application and approving the application using the Approve tool in the Application Details toolbar.
