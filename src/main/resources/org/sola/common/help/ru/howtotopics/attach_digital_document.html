Attach a Digital Document

To attach a digital document, use the Documents tab in the Application Details screen to add a new document to the application or select an existing document from the list and click the Attach button.



The File Browser dialog will display and you can use the file browser to select any digital document from your local computer file system or from the remote file location configured for SOLA.





The Remote disk view displays documents that have been placed in a specific network file system location e.g. The output folder configured for a network attached scanner. Thumbnail previews are supported for standard image formats (png, tiff, jpg) as well as PDF. Once you have chosen the document to attach, click the Attach button. A paperclip icon will be displayed in the documents list in the Documents tab to indicate the documents that have attachments. Finally you need to save the application for the attachment to remain linked to the application.

Note that moderate to large files can cause issues with the attachment process. For details, refer to Known Issues.
