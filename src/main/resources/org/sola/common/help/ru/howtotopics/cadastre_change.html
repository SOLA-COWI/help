Cadastre Change

The Cadastre Change screen allows you to subdivide/split or merge parcels. It can be accessed by starting a Cadastre Change service from the Application Details Service tab.



The Cadastre Change screen extends the SOLA Map Viewer with additional tools to aid spatial editing of parcels. The additional tools provided are:

  • Select / Deselect Target Parcel
  • Survey Points screen
  • Create / Link Survey Points
  • Create New Parcel
  • Parcel List screen
  • Select a boundary to change
  • Change selected boundary
  • Add image
  • Remove added image

Current functionality of the Cadastre Change is limited to simple parcels defined by a series of right lines, excluding natural boundaries defined by line segments.

Survey Points

Survey points represent coordinates recorded on a survey plan. They can be located directly on the map via a single click with the the Create / Link Survey Points tool or the coordinates can be manually entered or modified using the Survey Points screen. To remove a survey point, select the point in the Survey Points screen and click the Remove button.



The Survey Points screen can be used to indicate if the new survey point is a boundary or not (e.g. a traverse point). A boundary point can be optionally linked to the coordinate point of a target parcel using the Create / Link Survey Points tool. Simply select the new boundary point and drag it close to the target parcel coordinate point. The new boundary point will snap to the location of the target parcel coordinate point and the blue cross will change to a green cross to confirm the linkage. The Survey Points screen also indicates when the boundary point is linked and it can be used to unlink a boundary point by unchecking the Is linked check box.


Parcels

The existing parcels you wish to edit (i.e. the target parcels) can be identified using the Select / Deselect Target Parcel tool and/or by referencing the appropriate property on the Application Details Property tab. Target parcels are highlighted with a blue border. Once you have added and linked the necessary survey points, you can create new parcels using the Create New Parcel tool . Using the tool, single click each survey point of the new parcel in sequence in either a clockwise or anti-clockwise direction and double click on the final survey point to close the parcel boundary. The new parcel will display with a green border and yellow fill.



To edit the identifier for the new parcel, use the Parcel List tool to open the Parcel List screen and update the First and/or Last part of the parcel identifier. You can also use the Parcel List to remove a new parcel if you make a mistake while creating it.



Approving a Cadastre Change

Changes made to the cadastre remain pending until the application containing the cadastre change service is approved. You can approve the application by completing all services on the application and approving the application using the Approve tool in the Application Details toolbar. The new parcels will then show as current parcels in the SOLA Map Viewer.
